# Vue Admin Template

使用到的技术：
- vue
- vuex
- vux-router
- element-ui
- axios

## Build Setup
在`src/config/env.js`修改`BASE_URL`以及...
``` bash
# install dependencies
npm install

npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
